# Organ-donation-procurement_final-year-project
1.Project Title

Organ Donation Procurement System

2.Project Description

This project is a web-based application developed to manage and streamline the process of organ donation and transplantation. It helps in maintaining records of donors, patients, doctors, and organizations, and facilitates efficient matching and tracking of organ donations.

3.Objectives

Automate the manual organ donation process

Maintain centralized database of donors and recipients

Improve accuracy and reduce human errors

Enable faster organ matching and allocation

Increase awareness and efficiency in organ donation

4.Features

 User Registration & Login System
 
 Donor and Patient Management
 
 Doctor and Organization Management
 
 Search and View Patient/Donor Details
 
 Organ Matching and Transaction Tracking
 
 Basic Statistics Visualization
 
 Database Management using SQL
